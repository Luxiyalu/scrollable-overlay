How to: Prevent Body From Scrolling When Overlay Is On

Benefit of This Solution

  • Pure CSS
  • The content page won't skid to top with overlay on
  • Works on Mobile Safari

HTML Structure

    <div class="overlay">
        <div class="overlay-content"></div>
    </div>

    <div class="background-content">
        lengthy content here
    </div>

Styling

    html, body {
        height: 100%;
    }
                
    .overlay{
        position: fixed;
        top: 0px;
        left: 0px;
        right: 0px;
        bottom: 0px;
        background-color: rgba(0, 0, 0, 0.8);

        .overlay-content {
            height: 100%;
            overflow: scroll;
        }
    }
    
    .background-content{
        height: 100%;
        overflow: auto;
    }
